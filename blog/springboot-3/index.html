SpringBoot課程筆記(三)

By Jim Chien 2021-06-30

@Configuration

  • 用法：只能加在class上
  • 用途：Spring中的設定用註解，表示這個class是拿來設定Spring用的
@Configuration
public class MyConfiguraiotn(){
    //設定Spring
}

spring Boot 啟動 -> Spring去檢查class上的註解->

  1. Class上有 @Component 註解
  2. Class上有 @Configuration 註解
  3. Class上什麼註解都沒有(Spring容器不會理他)

->啟動成功

  • @component -> 將此class變成一個由Spring容器所管理的Bean
  • @Configuration -> 將此class變成設定Spring用的class

@Bean

  • 用法：只能加在帶有@Configuration class的方法上
  • 用途：在Spring容器中創建一個bean
@Configuration
public class MyConfiguraiotn(){
    //設定Spring

    @Bean
    public Printer myPrinter(){
        return new HpPrinter();
    }
}

這時Spring啟動時，Spring容器就會將HpPrinter new出來，取名為myPrinter。

@Configuration
public class MyConfiguraiotn(){
    //設定Spring

    @Bean("xxx")//這邊會優先當作bean的名稱
    public Printer myPrinter(){
        return new HpPrinter();
    }
}
